# Campo lampone github

This is the github organization profile for Campo-lampone code. Here are some functionalities of different projects under this organization:

## Projects
- [campo-image](https://github.com/CampoLampone/campo-image) - A fork of jetson-image repository that contains our custom Jetson Nano ubuntu image used in Jetbot robots
- [campo-fw](https://github.com/CampoLampone/campo-fw) - Repository containing firmware for traffic lights and raspberry pico used as motor controller on Jetbots. (Also contains PCB schematics for a board that manages connection between RPI pico and Jetbot)
- [campo-vis](https://github.com/CampoLampone/campo-vis) - React app managing visualisation for the whole Jetbot city/playground
- [campone](https://github.com/CampoLampone/campone) - Python library embedded into jetbots, used for controlling and managing the robots from high-level perspective (i.e. embedding basic computer-vision tasks from camera output, controlling robot using PID, etc.)
- [campone-host](https://github.com/CampoLampone/campone-host) - A bundle of shell scripts used to launch gstreamer connection between computer and Jetbot
- [campone-pytorch](https://github.com/CampoLampone/campone-pytorch) - ResNet training script that will be used for detection in future (not implemented yet)
- [JetbotMadness](https://github.com/CampoLampone/JetbotMadness) - Archived repository that consists of code used in CampoLampone 2024 (after that, we decided to scrap the whole code and start from scratch for the Campo-lampone 2025)
